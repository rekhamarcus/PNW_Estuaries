This repository contains all code used for data download and analysis for my MSc research project, which is focused on quantifying the effects of climate change on estuaries in the Pacific Northwest and incorporating this information into regional-scale conservation planning. This respository is organized into the following folders:
1. Scripts: all R code scripts for data analysis, including download, wrangling, modelling, analysis, and figure creation.
2. Functions: R code for specific functions which served as part of the data analysis.
3. AHP: datasets used for the creation of estuary conservation assessments, including priority matrices obtained from AHP surveys
4. Figures: all figures generated for publication and sharing
